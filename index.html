Introduction

What is wsqlite?

A production-grade SQLite ORM library engineered for modern Python applications. Built atop Pydantic's type validation system for type-safe database operations.

Type-Safe Database Operations

wsqlite leverages Pydantic models to define database schemas with full type safety. No SQL strings, no manual type conversions — just pure Python with automatic validation.

Perfect for data-intensive applications, microservices, and anywhere SQLite is the right tool for the job.

  • Type-Safe — Pydantic validation for all database operations
  • Async Support — Full async/await API for high-throughput scenarios
  • Connection Pooling — Built-in connection pool for performance
  • Auto Schema Sync — Models automatically sync with database
  • Production-Ready — 317 tests, 79% coverage, LTS release

Quick Start Guide

Start using wsqlite in minutes with these fundamental patterns.

Basic CRUD Operations

from pydantic import BaseModel
from wsqlite import WSQLite

class User(BaseModel):
  id: int
  name: str
  email: str

db = WSQLite(User, "users.db")

# Insert
user = db.insert(User(id=1, name="Alice", email="alice@example.com"))

# Query
all_users = db.get_all()
alice = db.get_by_id(1)

Async Operations

import asyncio
from wsqlite import WSQLite

async def main():
  db = WSQLite(User, "users.db")
  
  # Async insert
  await db.insert_async(User(id=2, name="Bob", email="bob@example.com"))
  
  # Async query
  users = await db.get_all_async()
  print(users)

asyncio.run(main())

Transactions

from wsqlite import Transaction

with Transaction("users.db") as tx:
  tx.execute("INSERT INTO users VALUES (?, ?, ?)", (3, "Charlie", "charlie@example.com"))
  tx.execute("INSERT INTO users VALUES (?, ?, ?)", (4, "Diana", "diana@example.com"))
  tx.commit() # Commits both or neither

Connection Pooling

from wsqlite import WSQLite, ConnectionPool

# With connection pooling
db = WSQLite(User, "users.db", pool_size=20)

# Or use pool directly
pool = ConnectionPool("users.db", pool_size=10)
conn = pool.get_connection()
