# Unit Tests

Core unit tests for wsqlite library components.

## Test Files

- `test_async.py` - Async operations tests
- `test_connection.py` - Database connection tests
- `test_exceptions.py` - Exception handling tests
- `test_query_builder.py` - Query builder tests
- `test_repository.py` - Repository pattern tests
- `test_sql_types.py` - SQL type mapping tests
- `test_sync.py` - Table sync tests
